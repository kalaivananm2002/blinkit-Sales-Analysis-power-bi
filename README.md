# blinkit-Sales-Analysis-power-bi
Blinkit Sales Analysis using Power BI to clean, model, and analyze sales data. Built an interactive dashboard to track KPIs like total sales, orders, AOV, category performance, time-based trends, and regional insights to support data-driven business decisions.

# Project Overview
This project focuses on analyzing Blinkit sales data using Power BI to uncover meaningful business insights. The goal is to track sales performance, understand product and category trends, and support data-driven decision making through an interactive dashboard.

# Objectives
-Analyze overall sales performance and order trends
-Identify top and underperforming product categories
-Track key KPIs such as total sales, orders, and average order value
-Understand time-based and regional sales patterns

# Tools & Technologies
Power BI
Power Query (Data Cleaning & Transformation)
DAX (Calculated Measures)
Data Modeling
Interactive Visualizations

# Dashboard Features
Key KPIs: Total Sales, Orders, AOV
Category-wise and product-wise sales analysis
Time-based trends (daily/monthly)
Regional sales distribution
Filters for date, category, and location

# Conclusion:
This project demonstrates the practical use of Power BI for sales data analysis, from data cleaning and modeling to building an interactive dashboard. The insights generated help understand sales performance, product trends, and demand patterns, supporting data-driven decision making. The project reflects real-world analytics workflows and showcases core data analyst skills.

Highlights sales trends and demand patterns
